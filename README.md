## Requirements  
Before setting up this terminal configuration, make sure the following tools are installed:  

- **Kitty** (Terminal emulator)  
- **Neovim (nvim)** (Text editor)  
- **Zsh** (Shell)  
- **Starship Prompt** (Cross-shell prompt)
